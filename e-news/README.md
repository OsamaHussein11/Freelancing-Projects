# Global News Network

A simple static news website built with HTML, CSS, and a small amount of JavaScript.

## Pages
- Home page
- Categories page
- Article pages
- Search page
- About page
- Contact page

## Features
- Responsive layout
- News cards written directly in HTML
- Category sections
- Simple search filtering with JavaScript
- Back-to-top button
- Contact page with message form and login form design

## Notes
This project is a front-end student project. It does not use PHP, a database, or an admin dashboard.
