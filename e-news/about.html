About Us

Global News Network is a student news website created to practice building clean and responsive web pages using HTML, CSS, and JavaScript.

The website includes a home page, category sections, article pages, search page, and contact page. The main goal is to organize news content in a clear way and make navigation simple for visitors.

Simple Layout

Cards, sections, and article pages are designed to be easy to read.

Responsive Design

The layout works on desktop and smaller screens.

Basic Interaction

JavaScript is used for search filtering and a back-to-top button.
